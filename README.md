# 🎯 MIFB — Million Items Filtering Backend & Frontend

список из 1,000,000 элементов с выбором, сортировкой и инфинити-скроллом.

## 📦 Версии

| Компонент | Версия | Последние изменения |
|-----------|--------|---------------------|
| **Бэкенд** | `1.4.0` | Ручной CORS, /api/version, все эндпоинты |
| **Фронтенд** | `0.8.0` | Левый список, виртуализация, поиск, футер с версией |

## 🚀 Быстрый старт

### Бэкенд
```bash
cd back
npm install
npm run dev  # http://localhost:3001
```

### Фронтенд
```bash
cd front
npm install
npm run dev  # http://localhost:5173
```

## 🌍 Деплой

- **Бэкенд**: [Render](https://mifb.onrender.com)
- **Фронтенд**: GitHub Pages (будет добавлено)

## 📡 API Endpoints

### Основные
- `GET /api/left?search=&offset=0&limit=20` — левый список
- `GET /api/right?search=&offset=0&limit=20` — правый список
- `POST /api/select` — выбрать элемент
- `POST /api/deselect` — отменить выбор
- `POST /api/reorder` — изменить порядок
- `POST /api/items` — добавить новый элемент
- `POST /api/batch` — пакетная обработка

### Служебные
- `GET /api/health` — состояние сервера
- `GET /api/version` — версия и коммит

## 📊 Версионирование

В футере фронтенда отображаются:
- Версия фронтенда (из `front/package.json`)
- Версия бэкенда (из `back/package.json`) + хеш коммита
- Окружение (production/development)
- Время последнего обновления

## 🛠 Технологии

### Бэкенд
- Express.js
- TypeScript
- Ручной CORS (никаких конфликтов!)

### Фронтенд
- React + TypeScript + Vite
- TanStack Query (кэширование, инфинити-скролл)
- TanStack Virtual (виртуализация)
- dnd-kit (Drag & Drop)
